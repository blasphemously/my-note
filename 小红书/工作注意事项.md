###### ark请求服务端接口设计

在package.json中 执行指令上更改服务器名 实现请求后端接口
```
"dev:app-item": "LOGAN_PROXY_KEY=ark-zhengyang1-sit lerna run --parallel --scope='ark-app-{root,layout,basics,item}' dev"

LOGAN_PROXY_KEY=ark-zhengyang1-sit // 设置后端接口
```
